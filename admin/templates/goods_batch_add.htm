  • Quy tắc làm file: Tải file mẫu tại đây
  • goods_sn: Mã SP. Nếu ko điền nó sẻ tạo tự động
  • brand_id: Mã ID của Thương hiệu sản phẩm, lấy ở trong phần thương hiệu ở đây
  • goods_name: Tên sản phẩm
  • market_price | shop_price: Giá thị trường | Giá Niêm Yết - Muốn thay đổi thì nhập còn không để trống
  • Để setup Khuyến mãi: is_promote = 1, is_promote = 0 là không khuyến mãi
  • Cột ngày định dạng yyyy-mm-dd, giử phím Ctrl rồi kéo góc phải ô để sao chép giá trị ngày.
  • Các Cột giá formart theo định dạng General tức là không phân cách hàng nghìn, trăm....
  • Hình ảnh: original_img - Size vuông: 400x400px hoặc 600x600px. Tạo thư mục lưu hình này ở cdn/images/yyyydd/source_img/. Ví dụ : cdn/images/202002/source_img/ten-san-pham.jpg, và điền vào file excel: images/202002/source_img/ten-san-pham.jpg
  • Hình Gallery: gallery_img - Size = với hình trên. Tạo thư mục lưu hình này ở cdn/images/yyyydd/source_img/. Ví dụ : cdn/images/202002/source_img/ten-san-pham-gallery.jpg, và điền vào file excel: images/202002/source_img/ten-san-pham-gallery.jpg
